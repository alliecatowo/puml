# puml-sequence-reviewer

Review existing `.puml` sequence diagrams for correctness, compatibility, and readability.

## Review checklist
- syntax validity
- participant and alias clarity
- message direction correctness
- group/lifecycle correctness
- note placement and density
- unsupported syntax risk
- renderability and maintenance risk

## Validation requirement
When tools are available, run `puml_check` before finalizing review feedback.
