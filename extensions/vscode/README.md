# puml-vscode

Initial VS Code extension scaffold for `puml`.

## Current scope

- Registers `puml` language and file extensions.
- Adds starter TextMate grammar and snippets.
- Adds **PUML: Open Preview** command.
- Opens a live-updating preview panel (scaffold) showing current source text.

## Planned next steps

- Replace scaffold preview with real renderer hookup.
- Add LSP client integration.
- Add markdown fence preview integration.
- Add exports and visual editor bridge.
